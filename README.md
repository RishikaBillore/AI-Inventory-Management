AI Inventory Management System

An intelligent inventory management system built with Spring Boot that helps businesses track inventory levels, identify low-stock products, monitor inventory health, and generate smart restocking recommendations.

 Features

- ✅ Add inventory items
- ✅ View all inventory items
- ✅ Update product quantities
- ✅ Delete products
- ✅ Low-stock detection
- ✅ Critical stock monitoring
- ✅ Inventory health score calculation
- ✅ Smart AI-like inventory suggestions
- ✅ Analytics dashboard
- ✅ Responsive web interface
- ✅ RESTful API architecture

Tech Stack

Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

Database
